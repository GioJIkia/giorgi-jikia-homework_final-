1.პროექტის სახელი და მოკლე აღწერა.

ა. მთავარი გვერდი — index.html (მთავარი გვერდი)
• Header (გვერდების ჩამონათვალი, დალინკვადი ლოგო და სანავიგაციო სტრუქტურა. კონტენტის განსალაგებლად გამოყენებულია flex, ელემენტებს აქვს hover ეფექტები).
• Section (ქავერ სურათით, ახალ კოლექციაზე გადამყვანი ღილაკით და ანიმაციით. კონტენტის განსალაგებლად გამოყენებულია flex, ელემენტებს აქვს hover ეფექტები).
• Section (H1 წარწერა და ქარდები. კონტენტის განსალაგებლად გამოყენებულია flex, ელემენტებს აქვს hover ეფექტები).
• Footer (ბრენდების და გვერდების ჩამონათვალი, მაღაზის საკონტაქტო ინფორმაცია და სოციალური მედია. კონტენტის განსალაგებლად გამოყენებულია flex, ელემენტებს აქვს hover ეფექტები).
